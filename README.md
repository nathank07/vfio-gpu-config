# VFIO GPU Config 
Simple script intended to switch between GPUs for VFIO passthrough on my system (Pop!_OS 22.04 LTS)
## Usage
Make swap.sh executable
```bash
chmod +x swap.sh
```
Run it with sudo
```bash
sudo ./swap.sh
```
## Disclaimer
This may not work on your system, you're free to fork it and adjust it to your needs.
